
# pokefetch

Quickly print a random Pokemon to your terminal along with your system information.


## Usage/Examples

```bash
# Run the script normally
$ pokefetch

# Just print the sprite and no other information
$ pokefetch -n

# Output script information to stderr
$ pokefetch -v
```
